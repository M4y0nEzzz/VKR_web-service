Техническое задание по проекту:
1) Одно мероприятие - одна запись в БД
2) Категории мероприятий:
      - общеуниверситетские,
      - внутри института (колледжа),
      - участие в других мероприятиях (организатор - не университет)
3) Название мероприятия - текст
4) Дата: время с ... по ...
5) Место проведения - таблица (возможно несколько мест проведения)
6) Подразделения - таблица
7) Ответственный - таблица
8) Комментарий, оно же описание - текст
9) Статус ПУБЛИКАЦИИ НА САЙТЕ УНИВЕРСИТЕТА - да/нет
10) Указание создателя записи (мероприятия)
11) Пользователь:
    - авторизация через LDAP,
    - использование группы безопасности,
    - ФИО, телефон,
    - почта,
    - подразделение
13) Роли:
    - пользователь (создает, удаляет и редактирует СВОИ мероприятия),
    - админ (права пользователя + управление подразделениями + управление пользователями + логирование)
14) Синхронизация с LDAP при каждом входе
15) Доступ по группе безопасности
